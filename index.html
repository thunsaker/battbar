pebble-battbar

A simple battery bar for add-in for Pebble developers.

Instructions

Add to my project

  1. Download Pebble-BattBar Source
  2. Copy battbar.h, battbar.c into your project "src" directory
  3. Copy all image files in resources/images to your project "resources/images" directory
  4. Declare the image resources in your appinfo.json. Add the following to the resource/media section
    {
                "type": "png",
                "name": "WHITE_LINE_FULL",
                "file": "images/white_line_tall.png"
              },
              {
              "type": "png",
              "name": "WHITE_LINE_LONG",
              "file": "images/white_line_long.png"
            },
            {
            "type": "png",
            "name": "BLACK_LINE_LONG",
            "file": "images/black_line_long.png"
          },
          {
          "type": "png",
          "name": "BLACK_LINE_FULL",
          "file": "images/black_line_tall.png"
        }
    Your appinfo.json should look something like this:
    {
        "versionLabel": "1",
        "shortName": "BattBar",
        "companyName": "thunsaker",
        ...
        "resources": {
        "media": [
        {
        "type": "png",
        "name": "WHITE_LINE_FULL",
        "file": "images/white_line_tall.png"
      },
      ...
      ]
    }
  }
  5. Include battbar.h in your primary c file (i.e. main.c)
  6. Setup BattBar Options
  7. Call DrawBattBar pass your BBOptions and current window layer

Example Usage

#include "pebble.h"
    #include "battbar.h" /* Step 4 */
    ...
    int main(void) {
    window = window_create();
    window_stack_push(window, true);
    
    Layer *window_layer = window_get_root_layer(window);
    GRect bounds = layer_get_frame(window_layer);

    ...

    /* Change these options to your liking */
    BBOptions options; /* Step 5 */
    options.position = BATTBAR_POSITION_TOP;
    options.direction = BATTBAR_DIRECTION_DOWN;
    options.color = BATTBAR_COLOR_BLACK;
    options.isWatchApp = true;

    DrawBattBar(options, window_layer); /* Step 6 */
    ...
  }

BattBar Options - BBOptions

position - enum BBPosition

  • BATTBAR_POSITION_RIGHT - BattBar on right of the screen
  • BATTBAR_POSITION_LEFT - BattBar on the left of the screen
  • BATTBAR_POSITION_BOTTOM - BattBar on the bottom of the screen
  • BATTBAR_POSITION_TOP - BattBar on the top of the screen

direction - enum BBDirection

  • BATTBAR_DIRECTION_UP - BattBar will shrink from the bottom of the screen toward the top
  • BATTBAR_DIRECTION_DOWN - BattBar will shrink from the top of the screen toward the bottom

color - enum BBColor

  • BATTBAR_COLOR_BLACK - BattBar will be black
  • BATTBAR_COLOR_WHITE - BattBar will be white

isWatchApp - bool

  • true - BattBar adjusts for the titlebar present in watch apps - 152px
  • false - BattBar will use the full height of the screen - 168px
